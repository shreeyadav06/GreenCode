# GreenCode Requirements (Project Environment)

This file documents runtime and tooling requirements for this repository.
It is not a Python pip requirements file.

Backend (Java API)
- Java JDK 25 (required)
- Jackson Databind 2.16.1
- Jackson Datatype JSR310 2.16.1
- Built-in Java HttpServer (`com.sun.net.httpserver`)

Frontend (Web App)
- Node.js 20+
- npm 10+
- Next.js 16.1.7
- React 19.2.3
- React DOM 19.2.3
- TypeScript 5+
- Tailwind CSS 4+
